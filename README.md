# Autumn 🍂

> Spring Boot-style web framework for Rust. Built on [Axum](https://github.com/tokio-rs/axum).

Autumn assembles proven Rust crates into a convention-over-configuration web
framework. It gives you route macros, HTML templating, database access, and a
Tailwind CSS build pipeline out of the box — so you can focus on your
application instead of wiring together middleware. If you've used Spring Boot,
Rails, or Laravel, the shape will feel familiar.

> **Autumn is v0.1 — experimental.** APIs will change. Not recommended for production use.

## Features

- **Route macros** — `#[get("/")]`, `#[post("/")]`, `#[put("/")]`, `#[delete("/")]`
- **Route collection** — `routes![handler1, handler2]` macro
- **Application bootstrap** — `#[autumn::main]` + `autumn::app().routes(...).run().await`
- **Configuration** — `autumn.toml` with `[server]`, `[database]`, `[log]`, `[health]` sections
- **Environment overrides** — `AUTUMN_SERVER__PORT`, `AUTUMN_DATABASE__URL`, etc.
- **Database pool** — Diesel + diesel-async + deadpool (Postgres)
- **Db extractor** — `db: autumn::Db` in handlers for an `AsyncPgConnection`
- **HTML rendering** — Maud (`autumn::html!`, `autumn::Markup`)
- **htmx integration** — auto-served at `/static/js/htmx.min.js`
- **Tailwind CSS** — build pipeline via `build.rs` + `autumn setup` CLI command
- **Health check** — auto-mounted JSON endpoint at `/health` (configurable)
- **Structured logging** — tracing with Pretty / JSON / Auto formats
- **Graceful shutdown** — Ctrl+C / SIGTERM with configurable drain timeout
- **Static file serving** — auto-served from `static/` directory
- **Error handling** — `AutumnError` with status refinement (`.not_found()`, `.bad_request()`, etc.)
- **JSON support** — `autumn::Json<T>` for request parsing and responses
- **Form support** — `autumn::extract::Form<T>` for form data
- **Path extraction** — `autumn::extract::Path<T>` for URL parameters

## Quickstart

```bash
# Install the CLI (from source — crates.io publication is not yet available)
cargo install --path autumn-cli

# Create a new project
autumn new my-app
cd my-app

# Download Tailwind CSS (optional — only needed for styled CSS)
autumn setup

# Run it
cargo run

# Visit http://localhost:3000
```

## Example

This is the `main.rs` generated by `autumn new`:

```rust
use autumn::{get, routes};

#[get("/")]
async fn index() -> &'static str {
    "Welcome to Autumn!"
}

#[get("/hello/{name}")]
async fn hello_name(name: autumn::extract::Path<String>) -> String {
    format!("Hello, {}!", *name)
}

#[autumn::main]
async fn main() {
    autumn::app()
        .routes(routes![index, hello_name])
        .run()
        .await;
}
```

## Built On

Autumn stands on the shoulders of these excellent crates:

- [Axum](https://github.com/tokio-rs/axum) — async web framework
- [Diesel](https://diesel.rs/) + [diesel-async](https://github.com/weiznich/diesel_async) — ORM and query builder (Postgres)
- [Maud](https://maud.lambda.xyz/) — compile-time HTML templating
- [htmx](https://htmx.org/) — HTML-driven interactivity
- [Tailwind CSS](https://tailwindcss.com/) — utility-first CSS
- [Tokio](https://tokio.rs/) — async runtime
- [Tracing](https://github.com/tokio-rs/tracing) — structured logging
