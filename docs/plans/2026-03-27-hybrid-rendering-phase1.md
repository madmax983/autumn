# Hybrid Rendering Phase 1 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Implement Phase 1 of hybrid rendering — `#[static_get]` macro, `StaticRouteMeta` types, `routes![]` integration, `StaticFileLayer` middleware, and blog example with an `/about` page.

**Architecture:** The `#[static_get]` macro reuses the existing `route_macro()` internals to generate the same `__autumn_route_info_{name}()` companion, plus a new `__autumn_static_meta_{name}()` companion returning `StaticRouteMeta`. The `routes![]` macro gains a sibling `__autumn_static_routes()` collector. At runtime, `AppBuilder::run()` detects a `dist/` directory and layers `StaticFileLayer` (wrapping `tower-http::ServeDir`) above the Axum router. Phase 1 scope: static paths only — no `params`, no `revalidate`, no `autumn build` CLI command yet.

**Tech Stack:** Rust, syn/quote/proc-macro2 (macros), tower-http ServeDir (middleware), serde_json (manifest), axum (runtime)

```rust
///

    let trimmed = url_path.trim_matches('/');
    if trimmed.is_empty() {
        "index.html".to_owned()

        format!("{trimmed}/index.html")
```

```rust
//! `#[static_get]` proc macro implementation.
//!
//! Generates the same route registration as `#[get]`, plus a companion
//! `__autumn_static_meta_{name}()` function returning `StaticRouteMeta`.

use proc_macro2::TokenStream;
use quote::{format_ident, quote};
use syn::{ItemFn, LitStr};

/// Parse `#[static_get("/path")]` and generate both the regular route
/// companion and a static metadata companion.
pub fn static_get_macro(attr: TokenStream, item: TokenStream) -> TokenStream {
    // Parse the path string from the attribute
    let path: LitStr = match syn::parse2(attr) {
        Ok(path) => path,
        Err(err) => return err.to_compile_error(),
    };

    // Validate path is not empty
    if path.value().is_empty() {
        return syn::Error::new(path.span(), "Route path must not be empty").to_compile_error();
    }

    // Validate path starts with '/'
    if !path.value().starts_with('/') {
        let suggested = format!("/{}", path.value());
        return syn::Error::new(
            path.span(),
            format!("Route path must start with '/'. Did you mean \"{suggested}\"?"),
        )
        .to_compile_error();
    }

    // Path must not contain path parameters (Phase 1: static paths only)
    if path.value().contains('{') {
        return syn::Error::new(
            path.span(),
            "static_get does not support path parameters yet. Use #[get] for parameterized routes.",
        )
        .to_compile_error();
    }

    // Parse the annotated function
    let input_fn: ItemFn = match syn::parse2(item.clone()) {
        Ok(f) => f,
        Err(_) => {
            return syn::Error::new_spanned(
                item,
                "static_get can only be applied to functions",
            )
            .to_compile_error();
        }
    };

    // Must be async
    if input_fn.sig.asyncness.is_none() {
        return syn::Error::new_spanned(
            input_fn.sig.fn_token,
            "Autumn route handlers must be async functions",
        )
        .to_compile_error();
    }

    let fn_name = &input_fn.sig.ident;
    let route_info_name = format_ident!("__autumn_route_info_{}", fn_name);
    let static_meta_name = format_ident!("__autumn_static_meta_{}", fn_name);
    let vis = &input_fn.vis;

    quote! {
        #input_fn

        #[doc(hidden)]
        #vis fn #route_info_name() -> ::autumn_web::route::Route {
            ::autumn_web::route::Route {
                method: ::autumn_web::reexports::http::Method::GET,
                path: #path,
                handler: ::autumn_web::reexports::axum::routing::get(#fn_name),
                name: ::core::stringify!(#fn_name),
            }
        }

        #[doc(hidden)]
        #vis fn #static_meta_name() -> ::autumn_web::static_gen::StaticRouteMeta {
            ::autumn_web::static_gen::StaticRouteMeta {
                path: #path,
                name: ::core::stringify!(#fn_name),
                revalidate: ::core::option::Option::None,
            }
        }
    }
}
```

The `routes![]` macro currently emits `vec![__autumn_route_info_a(), ...]`. It needs to ALSO emit a function that collects static metadata. However, `routes![]` doesn't know which handlers are `#[static_get]` vs `#[get]` at macro expansion time — it only sees names.

**Design decision:** Instead of modifying `routes![]`, add a separate `static_routes![]` macro that mirrors `routes![]` but calls `__autumn_static_meta_{name}()` instead. This is cleaner — only list the static route names in `static_routes![]`.

Actually, re-reading the design doc: the `routes![]` macro is supposed to generate a `__autumn_static_routes()` function. But at macro expansion time, `routes![]` can't know which handlers have static companions. The cleanest Phase 1 approach: the `__autumn_static_meta_*` functions exist silently. The build system (Phase 2+) or runtime discovers them through a separate mechanism.

For Phase 1, we don't need `routes![]` changes. The `StaticFileLayer` middleware works purely off the `dist/manifest.json` file. The macro just needs to generate the companions; they're used by `autumn build` (future) and compile-pass tests verify they exist.

**Skip this task** — `routes![]` changes are deferred to when `autumn build` CLI is implemented. The manifest-driven middleware doesn't need them.

```rust
//! Tower middleware that serves pre-rendered static HTML from a `dist/` directory.
//!
//! Wraps `tower-http::ServeDir` to get correct Content-Type, conditional
//! request handling, HEAD support, and range requests for free. Adds only
//! manifest-aware path mapping on top.

use axum::response::Response;
use http::Request;
use std::path::{Path, PathBuf};
use std::sync::Arc;
use std::task::{Context, Poll};
use tower::Service;
use tower_http::services::ServeDir;

use super::StaticManifest;

/// Layer that wraps the Axum router to serve static files from `dist/`.
///
/// For any incoming GET request whose path matches a route in the manifest,
/// the middleware attempts to serve the corresponding file from `dist/`.
/// If the file exists, it's served directly (bypassing the router).
/// If not, the request falls through to the Axum router for SSR.
#[derive(Clone)]
pub struct StaticFileLayer {
    dist_dir: PathBuf,
    manifest: Arc<StaticManifest>,
}

impl StaticFileLayer {
    /// Create a new `StaticFileLayer` serving from the given directory.
    ///
    /// Returns `None` if the manifest file doesn't exist or can't be parsed.
    pub fn new(dist_dir: impl Into<PathBuf>) -> Option<Self> {
        let dist_dir = dist_dir.into();
        let manifest_path = dist_dir.join("manifest.json");

        if !manifest_path.exists() {
            return None;
        }

        let manifest = StaticManifest::load(&manifest_path).ok()?;

        Some(Self {
            dist_dir,
            manifest: Arc::new(manifest),
        })
    }

    /// Returns the loaded manifest (for startup reconciliation logging).
    pub fn manifest(&self) -> &StaticManifest {
        &self.manifest
    }

    /// Returns the dist directory path.
    pub fn dist_dir(&self) -> &Path {
        &self.dist_dir
    }

    /// Check if a given request path has a static file in dist/.
    ///
    /// Returns the filesystem path if the file exists.
    pub fn resolve(&self, request_path: &str) -> Option<PathBuf> {
        // Check if this path is in the manifest
        let entry = self.manifest.routes.get(request_path)?;

        // Build the filesystem path
        let file_path = self.dist_dir.join(&entry.file);

        // Only return if the file actually exists
        if file_path.exists() {
            Some(file_path)
        } else {
            None
        }
    }
}
```

```rust
/// Build the router, optionally layering static file serving from `dist_dir`.
pub(crate) fn build_router_with_static(
    route_list: Vec<Route>,
    config: &AutumnConfig,
    state: AppState,
    dist_dir: Option<&std::path::Path>,
) -> axum::Router {
    let app_router = build_router(route_list, config, state);

    // If dist/ exists with a manifest, layer static serving on top
    if let Some(dist) = dist_dir {
        if let Some(layer) = crate::static_gen::StaticFileLayer::new(dist) {
            // Log startup reconciliation
            for (path, _entry) in &layer.manifest().routes {
                tracing::info!(path = %path, "Static route registered from dist/");
            }

            // ServeDir with fallback to the app router.
            // Static files take priority; unknown paths fall through to SSR.
            let serve_dir = tower_http::services::ServeDir::new(dist)
                .fallback(app_router);

            return axum::Router::new()
                .nest_service("/", serve_dir);
        }
    }

    app_router
}
```
