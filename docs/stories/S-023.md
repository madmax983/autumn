## User Story

As a **framework user**
I want to **use `Form<T>` in my handler signatures to extract form-submitted data**
So that **I can handle HTML form submissions without depending on axum directly**

---

## Description

### Background

Axum provides `Form<T>` for extracting URL-encoded form data. Autumn re-exports it so users only need `use autumn_web::extract::Form` (or eventually `use autumn_web::prelude::*`). Since `Form<T>` already implements `FromRequest`, this story is primarily about re-exporting and testing.

The `extract` module already exists with `Path` and `Query` re-exports (from S-006). This story adds `Form` to the same module.

After this story:
```rust
use autumn_web::{post, extract::Form, AutumnResult};
use serde::Deserialize;

#[derive(Deserialize)]
struct LoginForm {
    username: String,
    password: String,
}

#[post("/login")]
async fn login(Form(data): Form<LoginForm>) -> AutumnResult<&'static str> {
    // data.username, data.password available
    Ok("logged in")
}
```

### Scope

**In scope:**
- Re-export `axum::extract::Form` from `autumn_web::extract`
- Compile-pass test: handler with `Form<T>` parameter compiles
- Integration test: POST with form data, verify extraction works
- Document `Form<T>` in the extract module

**Out of scope:**
- Custom form validation (user-land concern)
- Multipart form data (separate extractor, v1.0)
- Custom rejection for form parse errors (use Axum's default for now)

## Technical Notes

### Implementation

One line in `autumn/src/extract.rs`:

```rust
pub use axum::extract::{Form, Path, Query};
```

That's it. `Form<T>` already implements `FromRequest` in Axum, and Axum's `Form` rejection returns 422 Unprocessable Entity by default.

### Testing

```rust
#[tokio::test]
async fn form_extraction_works() {
    use axum::Router;
    use axum::body::Body;
    use axum::http::{Request, StatusCode};
    use autumn_web::extract::Form;
    use serde::Deserialize;
    use tower::ServiceExt;

    #[derive(Deserialize)]
    struct TestForm {
        name: String,
        age: u32,
    }

    async fn handler(Form(data): Form<TestForm>) -> String {
        format!("{} is {}", data.name, data.age)
    }

    let app = Router::new()
        .route("/", axum::routing::post(handler));

    let response = app
        .oneshot(
            Request::builder()
                .method("POST")
                .uri("/")
                .header("content-type", "application/x-www-form-urlencoded")
                .body(Body::from("name=Alice&age=30"))
                .unwrap(),
        )
        .await
        .unwrap();

    assert_eq!(response.status(), StatusCode::OK);
    let body = axum::body::to_bytes(response.into_body(), usize::MAX)
        .await
        .unwrap();
    assert_eq!(&body[..], b"Alice is 30");
}
```
